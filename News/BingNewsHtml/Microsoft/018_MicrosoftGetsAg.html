Microsoft Gets Aggressive, Now Warns You If You're Running Windows 11 on Incompatible Hardware

Extreme Tech
Mon, December 2, 2024 at 6:05 PM UTC
1 min read

As the end-of-life date for Windows 10 approaches, Microsoft has updated its guidance for users running Windows 11 on incompatible hardware. The company added a new disclaimer and watermark to warn users about possible risks and limits.

Microsoft acknowledges that some users are upgrading to Windows 11 on unsupported devices. A new disclaimer warns users about potential compatibility issues and lack of future support, XDA reports.

The company said PCs running Windows 11 on incompatible hardware will no longer receive updates, and manufacturer warranties will not cover damage due to compatibility issues. An updated support page now includes this statement:

This PC doesn't meet the minimum system requirements for running Windows 11 - these requirements help ensure a more reliable and higher quality experience. Installing Windows 11 on this PC is not recommended and may result in compatibility issues.

After Oct. 14, 2025, Windows 10 will no longer receive security updates or technical support. Using Windows past this date could lead to security risks for users.

With the deadline approaching, users with incompatible hardware must decide whether to upgrade their devices, pay Microsoft for an extra year of support, try a different OS, or risk using unsupported software.
